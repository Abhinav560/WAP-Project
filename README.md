Project Overview
This project is a web application built using JavaScript where I am working with a public API. The main idea is to fetch anime quotes (Attack on Titan) and display them on a webpage.

The app will include features like search, filtering, and sorting so that the data is easier to use. I will also try to integrate Google Analytics to track basic user activity.

Features:

* Fetch data from API using fetch
* Display quotes on the page
* Search functionality
* Filter quotes by character
* Sorting options
* Responsive design

Bonus Features:

* Pagination
* Loading indicator

APIs Used:

* Attack on Titan Quotes API
* Google Analytics

Technologies Used:

* HTML
* CSS
* JavaScript
* Fetch API

How to Run:
Clone the repository and open the index.html file in a browser.

Project Structure:
index.html
style.css
script.js
README.md

Note:
Array methods like filter, map, and sort will be used instead of loops where required.
The project is being developed step by step according to the milestones.
